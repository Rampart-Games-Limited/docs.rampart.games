# Faq

<details>

<summary>How do I fix performance issues ?</summary>

Try:

* Lowering player cap
* Reducing AI/creature counts
* Selecting a smaller or less intensive map
* Upgrading CPU/RAM resources
* Restarting the server regularly

</details>
