# Setting a server name

<figure><img src="../../.gitbook/assets/Valheim server name.gif" alt=""><figcaption></figcaption></figure>

1. Click the "Startup" tab on the left of the control panel
2. Scroll down to "Server Name", and input your desired name
3. Click "Restart" at the top of the control panel

Note: The server name defines which world save file is loaded.  If you want to continue with an existing save, the server name must match the name of the .db and .fwl files in .config/unity3d/IronGate/Valheim/worlds\_local
